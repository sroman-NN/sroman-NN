# Sebastián G. de Jesús Romanelli (`sroman-nn`) 👋

I have been programming applications and video games, as well as various services and tools, for 9 years. My development preferences always lean towards **optimization, low-level development**, and, of course, absolute control over each dependency, library, or system as much as possible. 

However, I also have a considerable capacity for adaptation; if I come up with a project, I simply get to work and use whatever language, framework, or tool can help me bring it to life.

> ⚡ **My number one inspiration for programming is anger.** I hate when things fail, when they're inefficient, and of course, when they don't make sense.
